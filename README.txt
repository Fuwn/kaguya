kaguya - Fuwn's Anime & Manga Utility

Usage:
  kaguya COMMAND
  kaguya [COMMAND] --help | -h
  kaguya --version | -v

Commands:
  anichart      
  myanimelist   
  anilist        (default)
  crunchyroll   
  hidive        
  watch         
  character     
  x             

Options:
  --help, -h
    Show this help

  --version, -v
    Show version number
